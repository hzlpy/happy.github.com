C++ 知识点记录

发表于 2015-08-06   |     |   阅读次数   |  
  1. #include <> 只搜索系统目录；#include ""先搜索本地目录；如果在本地目录没有找到，再搜索系统目录；
  1. 深拷贝和浅拷贝可以简单地理解为：如果一个类拥有资源（堆，或者是其他系统资源），当这个类的对象发生复制过程的时候，这个过程可以叫做深拷贝；反之，对象存在资源，但复制过程并未复制资源的情况称为浅拷贝。
  2. 如果类使用 struct 保留字定义的，则在第一个访问标号之前的成员是公有的 (public)；如果类似用 class 保留字定义的，则这些成员是私有的 (private)。使用 struct 保留字定义的类默认具有 public 继承；使用 class 保留字定义的类默认具有 private 继承。
  3. 引用与指针的区别
    • 引用必须初始化，而指针不必；
    • 引用初始化之后不能被改变，而指针可以改变所指的对象；
    • 不存在指向空值的引用，而存在指向空值的指针；

  4. 以下三种情况下需要使用初始化成员列表

    • 需要初始化的数据成员是对象；
    • 需要初始化 const 修饰的类成员；
    • 需要初始化引用类型的成员数据；

  5. C++ 多态的实现原理
    多态分为类的多态性与函数的多态性。类的多态是通过虚函数与动态绑定实现的，函数的多态是通过重载实现的。
    存在虚函数的类都有一个一维的虚函数表叫做虚表。类的对象有一个指向虚表开始的指针。虚表是和类对应的，虚表指针是和对象对应的。

  6. C++ static 关键字

    • 在类中的 static 成员变量属于整个类所拥有，对类的所有对象只有一份拷贝；
    • 在类中的 static 成员函数属于整个类所拥有，这个函数不接收 this 指针，因而只能访问类的 static 成员变量；静态成员函数可以继承和覆盖,但无法是虚函数；
    • static 修饰的变量放在静态存储区，所以它具备持久性和默认值0，生命周期是程序的生命周期；（static 局部变量的“记忆性”与生存周期的“全局性”）

  7. C++ const 关键字

    • 可以用来定义 const 常量。 const int MAX = 100;
    • 便于进行类型检查。const 常量拥有数据类型，而宏常量没有数据类型。
    • 可以保护被修饰的变量。防止意外修改，增强程序的健壮性。
    • 为函数重载提供了一个参考。
