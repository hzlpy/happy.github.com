利用Hexo搭建博客

发表于 2014-11-19   |     |   阅读次数

准备

  1. 安装git客户端
  2. 注册github账号
  3. 安装Nodejs

开始

  1. 安装Hexo
    在CMD中输入命令：

    npm install -g hexo

    完成后，请注意提示。依照提示，再输入命令：

     npm install

  2. 创建文件夹
    例如，在F盘下创建一个hexo文件夹，存放博客相关的文件。进入hexo文件夹，在此文件夹下打开git bush，输入命令：

     hexo init

    hexo 文件夹下会出现一些文件和文件夹。

  3. 生成一套静态网页
    继续在git bash中输入命令：

     hexo generate 

     hexo server

    输入完毕后，会提示你使用浏览器进入http://localhost:4000 。

完成

使用浏览器进入http://localhost:4000 ，查看效果。

参考

  2. hexo你的博客
  3. 使用 Hexo 生成一套静态博客网页
