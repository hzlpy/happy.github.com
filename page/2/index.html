第7章 函数

发表于 2015-03-29   |     |   阅读次数

本章介绍了函数的声明和定义。主要围绕以下两个方面进行：

  1. 介绍怎样给函数传递参数以及函数怎样返回值；
  2. 介绍函数中的特殊情况：内联函数、类成员函数、重载函数；

参数传递

形参的初始化与变量的初始化一样。如果形参是非引用类型，则复制实参的值即实参的副本；如果形参为引用类型，则它只是实参的别名。

函数返回值

不能返回局部变量的引用。当函数执行完后，将释放局部变量的存储空间。和返回局部变量的引用一样，返回指向局部变量的指针也是错误的。函数执行完，局部变量的存储空间释放，返回的指针变成了指向不再存在的对象的垂悬指针。

函数中的特例

内联函数是为了避免函数调用的开销。它的写法是在函数返回类型前面添加关键字inline。
构造函数是特殊的类成员函数。与其它成员函数不同的是：构造函数和类同名，且无返回类型。
出现在相同作用域的两(多)个函数，如果具有相同名字而形参表不同则称为重载(overload)函数。

重载确定的三个步骤：

  1. 候选函数：与被调用函数同名的函数，并且在调用点上它的声明可见；
  2. 选择可见函数：从候选函数中选择一个或者多个函数，它们能够用指定的实参来调用；
  3. 寻找最佳匹配。。

求长度的那些事

发表于 2015-03-05   |     |   阅读次数

首先看我遇到的一道题目。

//题目：下面程序的输出结果为多少？

void foo(char str_arg[100])
{
    printf("%d \n", sizeof(str_arg));
}

int main(void)
{
    char str[] = "Hello";
    printf("%d \n", sizeof(str));
    printf("%d \n", strlen(str));
    char *p = str;
    printf("%d \n", sizeof(p));
    foo(str);
}

正确的答案是：6 5 4 4 。
下面具体分析一下。

  1. 我们知道 char str[] = "Hello"; 与 char str[] = {'H', 'e', 'l', 'l', 'o', '\0'}; 是等价的。前者为后者的语法糖。知道这一点，就容易得到 sizeof(str) 为 6。
  2. strlen(str)表示返回字符串 str 中字符的个数，显然 strlen(str) 为 5 。
  3. 在 32 位系统中，指针的长度都是 4 。故，sizeof(p) 为4。
  4. foo 函数中 str_arg 只表示一个指针，同上。

The length of a C string is determined by the terminating
null-character: A C string is as long as the number of characters
between the beginning of the string and the terminating null character
(without including the terminating null character itself).

HelpMe开发记录二

发表于 2015-01-26   |     |   阅读次数

在完成HelpMe的时候遇到了一个问题：假定知道一顿饭可用的花销cost与菜单上所有菜的单价price。在花销范围以内，怎样点菜使得可吃的菜最多？当然了每种菜限点一份了。（后面简称为点菜问题）
想了一下，发现这个问题的解法类似0-1背包问题。

0-1背包问题：
有N件物品和一个容量为Capacity的背包。每种物品均只有一件,第i件物品的重量是W[i]，价值是V[i]。求解将哪些物品装入背包可使价值总和最大。

在点菜问题中，花销cost相当于背包的容量Capacity，每份菜的花销相当于待放入背包中的物品的重量，最后点得的菜的数量的最大化相当于背包中物品价值的最大化。

表

如果依据存储结构来划分表，可以划分为顺序存储结构的顺序表和链式存储结构的链表。

表的常见操作有：

  1. 创建表；
  2. 遍历表；
  3. 判断表是否为空；
  4. 删除指定的表中元素；
  5. 插入指定的元素到表；

顺序表

用顺序存储方法存储的线性表被称为顺序表(Sequential List)。
我们可以这样用C语言定义一个顺序表。

#define size 10
typedef int DataType;
typedef struct {
    DataType data[size];
    int length;         //当前表的长度
}SeqList;

链表

用链式存储方式存储的线性表被称为链表(Linked List)。
链表的结点结构

链表中存放的结点包含两个域：数据域和指针域。数据域存放的是当前结点的数值(data)。指针域存放的是下一个结点的地址。

字符串与字符串函数

发表于 2014-11-30   |     |   阅读次数

字符串数组的定义

char str[10]; //定义一个字符串数组

字符串数组的初始化

常见的字符串初始化方法有以下几种：

1.char str[20] = "Hi, I am Happy."; 

2.char str[] = "Hi, I am Happy.";

3.char str[] = {'H','i',',',' ','I',' ','a','m',' ','H','a','p','p','y','\0'};

4.char *str = "Hi, I am Happy.";

方法2其实是3的语法糖。
方法1中多余地添加上元素个数，容易出现错误。比如：

char str[3] = "abc"; //元素个数应该为4

这里忘记了’\0’的存在。为了避免这样的错误，应该省略元素个数的定义，把对元素计数的工作交给编译器来做。

方法2与方法4的不同在于：相对前者初始化”char 的数组”的内容，后者是利用字符串常量初始化”指向 char 的指针”。字符串常量一般保存在只读的内存区域。《征服C指针》 6.2.7 char 数组的初始化

字符串数组与指针

1.字符串数组

在数组形式str[]中，str是一个地址常量。str不能被修改。可以使用运算符str + 1来表示数组里的下一个元素。然而，不能使用 ++str表示数组里的下一个元素，因为增量运算符只能用在变量前，而不能使用在常量前面。

2.指针

指针形式*str在程序开始执行之后，还需要为指针变量str预留一个存储位置，用来存储字符串的地址。这个变量初始时指向字符串的第一个字符，但是它的值时可以被改变的，可以对它使用增量运算符。

字符串输入与输出

1.字符串输入

1. gets(); 
2. fgets(); 
3. scanf();

2.字符串输出

1. puts(); 
2. fputs(); 
3. printf();

常见的字符串函数

1. strlen(); 
2. strcat(); 
3. strncat(); 
4. strcmp(); 
5. strncmp(); 
6. strcpy();
7. strncpy();

一维数组与指针

//求数组中元素之和
int sum(int a[], int n)
{
    int i = 0;                  //循环标记
    int total = 0;              //和
    for ( i = 0; i < n; i++) {    //求和
        total += a[i];
    }
    printf("The size of a is %u bytes. \n", sizeof(a));
    return total;                //返回 和
}

上面代码表示对数组元素求和，利用数组的下标取数组中每个元素的值。

//求数组元素之和
int sump(int *start, int *end)
{
int total = 0;                    //和
while (start < end) {
    total += (*start);          //把值累加的total上
    start++;                    //把指针指向下一个元素
    }
    return total;
}

上面代码表示对数组元素求和，利用指针取数组中每个元素的值。

二维数组与指针

//统计所有年份的总降水量
float getYearsTotalRainfall(float rain[][MONTHS])
{
    int y,m;    //循环标记
    float total = 0;
    float temp;
    for (y = 0; y < YEARS; y++) {
        for (m = 0; m < MONTHS; m++) {
            //temp = rain[y][m];
            //temp = *(&rain[0][0] + y*MONTHS + m);
            temp = *(*(rain+y)+m);
            total += temp;
        }
    }
    return total;
}

float getYearsTotalRainfall2(float *prain, int years, int months)
{
    int y,m;    //循环标记
    float total = 0;
    float temp;
    for (y = 0; y < years; y++) {
        for (m = 0; m < months; m++) {
            //temp = rain[y][m];
            //temp = *(&rain[0][0] + y*months + m);
            temp = *prain++;
            total += temp;
        }
    }
    return total;
}

在代码中列出了3种基本取二维数组(i,j)处的值的表示方法

temp = rain[y][m];                      //1. 用下标取(i,j)处的值
temp = *(&rain[0][0] + y*MONTHS + m);   //2. 用指针来取值
temp = *(*(rain+y)+m);                  //3. 用指针来取值

通过地址来引用二维数组元素若有以下定义：
int a[3][4],i,j;且当0≤i＜3、0≤j＜4，则a数组元素可用以下五种表达式来引用：

  1. a[i][j]
  2. *(a[i]+j)
  3. ((a+i)+j)
  4. (*(a+i))[j]
  5. (&a[0][0]+4i+j)

利用Hexo搭建博客

发表于 2014-11-19   |     |   阅读次数

准备

  1. 安装git客户端
  2. 注册github账号
  3. 安装Nodejs

开始

  1. 安装Hexo
    在CMD中输入命令：

    npm install -g hexo

    完成后，请注意提示。依照提示，再输入命令：

     npm install

  2. 创建文件夹
    例如，在F盘下创建一个hexo文件夹，存放博客相关的文件。进入hexo文件夹，在此文件夹下打开git bush，输入命令：

     hexo init

    hexo 文件夹下会出现一些文件和文件夹。

  3. 生成一套静态网页
    继续在git bash中输入命令：

     hexo generate 

     hexo server

    输入完毕后，会提示你使用浏览器进入http://localhost:4000 。

完成

使用浏览器进入http://localhost:4000 ，查看效果。

  1. 如何搭建一个独立博客——简明Github Pages与Hexo教程
  2. hexo你的博客
  3. 使用 Hexo 生成一套静态博客网页

平台：Windows 7 x64
