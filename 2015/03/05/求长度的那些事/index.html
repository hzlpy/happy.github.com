求长度的那些事

发表于 2015-03-05   |     |   阅读次数   |  

首先看我遇到的一道题目。

//题目：下面程序的输出结果为多少？

void foo(char str_arg[100])
{
    printf("%d \n", sizeof(str_arg));
}

int main(void)
{
    char str[] = "Hello";
    printf("%d \n", sizeof(str));
    printf("%d \n", strlen(str));
    char *p = str;
    printf("%d \n", sizeof(p));
    foo(str);
}

正确的答案是：6 5 4 4 。
下面具体分析一下。

  1. 我们知道 char str[] = "Hello"; 与 char str[] = {'H', 'e', 'l', 'l', 'o', '\0'}; 是等价的。前者为后者的语法糖。知道这一点，就容易得到 sizeof(str) 为 6。
  2. strlen(str)表示返回字符串 str 中字符的个数，显然 strlen(str) 为 5 。
  3. 在 32 位系统中，指针的长度都是 4 。故，sizeof(p) 为4。
  4. foo 函数中 str_arg 只表示一个指针，同上。

The length of a C string is determined by the terminating
null-character: A C string is as long as the number of characters
between the beginning of the string and the terminating null character
(without including the terminating null character itself).
