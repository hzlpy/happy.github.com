在安装 Flume 1.7.0 之前，先要了解它对系统的要求：

  1. Java 运行环境 Java 1.7 或更高版本
  2. 足够的内存来配置 sources, channels 和 sinks
  3. 足够的硬盘空间来配置 channels 和 sinks
  4. agent 有用到的目录的读/写权限
